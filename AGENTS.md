# Skills

* `mix format --check-formatted` - Check if the code is formatted according to the project's formatting rules.
* `mix format` - Format the code according to the project's formatting rules.
* `mix test` - Run the project's test suite to ensure that all tests pass.
* `mix test FILENAME` - Run the tests in the specified file.
* `mix deps.get` - Fetch and update the project's dependencies.
* `mix compile` - Compile the project's source code.
